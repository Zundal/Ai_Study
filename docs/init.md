# 머신러닝 딥러닝

1. 인공지능이란
    - 사람처럼 학습하고 추론 할 수 있는 지능을 가진 컴퓨터
    - 1967 프랑크 로젠 블라트 -> 퍼셉트론 발표
    - 영화속 인공지능의 종류 "인공 일반지능 [Antificial general inteligence]" "경 인공지능 [Strong]"
    - 사람과 구분하기 어려운 지능을 가진 컴퓨터 시스템 === 인공 일반지능
    - 우리가 마주치는 인공지능 "약 인공지능 [week]" === 특정 분야의 특정기능을 보조하는 역할

---

2. 머신러닝이란
    - 머신러닝은 규칙을 일일이 프로그래밍 하지 않아도 자동으로 데이터에서 규칙을 학습하는 알고리즘을 연구하는 분야.
    - 머신러닝은 통계학과 깊은 관련이 있다.
    - 대표적인 머신러닝 라이브러리 "사이킷 런 [sikit-learn]"
    - 사이킷 런은 파이썬 api 사용 

---

3. 딥러닝이란
    - 많은 머신러닝 알고리즘 중에 인공 신경망을 기반으로 한 방법들을 통칭하여 딥러닝 이라고 부릅니다.
    - 종종 사람들은 인공 신경망과 딥러닝을 크게 구분하지 않고 사용한다.
    - 2015년 구글은 딥러닝 라이브러리 텐서플로를 오픈소스로 공개
    - 페이스북은 2018년 파이토치 딥러닝 라이브러리를 오픈소스로 발표

- 인공지능
  - 사람처럼 학습하고 추론하는 지능
- 머신러닝
  - 규칙을 프로그래밍하지 않아도 자동으로 데이터에서 규칙을 학습하는 알고리즘 연구 분야
- 딥러닝
  - 인공 신경망이라고도 하고 텐서플로, 파이토치가 대표적 라이브러리
